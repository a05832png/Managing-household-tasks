# 🏠 ניהול הבית המשותף שלנו

ברוכים הבאים למאגר הניהול הביתי! המטרה של המאגר הזה היא לרכז את כל המשימות, התוכניות והרעיונות שלנו במקום אחד מסודר, עם סנכרון ישיר ל-**Google Chat**.

## 🚀 פונקציות עיקריות
* **ניהול משימות:** מעקב אחרי תיקונים, קניות וסידורים.
* **תיעוד:** שמירת מידע חשוב שצריך לשלוף מהר.
* **אוטומציה:** קבלת התראות לצאט בכל פעם שמתעדכן נושא חשוב.

## 🛠️ איך להשתמש?
1. **הוספת משימה:** פותחים `Issue` חדש ב-GitHub.
2. **דיונים:** משתמשים בתיאור של ה-Issue כדי להתייעץ אחד עם השני.
3. **סגירת משימה:** כשמסיימים, סוגרים את ה-Issue והוא נעלם מהרשימה הפתוחה.

## 🤖 חיבור ל-Google Chat
המאגר מחובר באמצעות Webhook לחדר הצ'אט המשותף. 
- עדכונים על משימות דחופות יופיעו שם באופן אוטומטי.
